# 橙4 + 蓝3

![橙4 + 蓝3](../images/variations/easy-4-open-closed.jpg)

## 概述

[橙4](../rolls/easy-4.zh.md#橙轮) + [蓝3](../rolls/closed-open-open-closed.zh.md#蓝轮)开头很简单，可以在橙轮连续跳4个坑或在蓝轮连续跳3个坑。该版型在第9圈粘液涨至最高后会变得较难。
