# 橙多挡板 + 蓝天坑

![橙多挡板 + 蓝天坑](../images/variations/5-waller-grand-canyon.jpg)

## 概述

[橙多挡板](../rolls/5-waller.zh.md) + [蓝天坑](../rolls/grand-canyon.zh.md)经常被认为是第三难的版型。

3. 继续留在橙轮直到可以切回蓝轮

## 路线推荐

以下是一个三人平冠的路线选择示例：

* 一名玩家在第9~12圈走蓝起路线
* 一名玩家在第9~12圈走天坑细边路线
* 一名玩家在第9~12圈走半侧跳路线
