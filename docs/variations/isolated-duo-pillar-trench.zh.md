# 橙浮岛 + 蓝多挡板

![橙浮岛 + 蓝多挡板](../images/variations/isolated-duo-pillar-trench.jpg)

## 概述

[橙浮岛](../rolls/isolated-duo.zh.md#橙轮) + [蓝多挡板](../rolls/pillar-trench.zh.md)的路线选择比较自由。由于橙轮浮岛后的坑为大坑，蓝轮上的空间相对更多一些。由于橙轮和蓝轮结尾处均有挡板，选择路线时注意给其他玩家留出空间。
