# 避开闸总

## 学习滚轮版型

熟悉滚轮版型会让你能够随时变换路线以避开闸总。

## 不要过早吸引注意

远离闸总且不要给自己吸引注意。更多地关注滚轮版型和路线规划。很多闸总花太多精力闸人导致他们没有关注滚轮版型，以致后期粘液上升、滚轮加速后，他们很快就出局了。
