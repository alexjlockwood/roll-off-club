# 浮岛版型跳大坑

橙蓝[浮岛](../rolls/isolated-duo.zh.md)版型均包含一个比较容易跳的中坑和一个后期比较难跳的大坑。

想要跳过大坑，你须要：

* 把握好起跳时机，在最高点稍微靠后、也就是略上坡的位置起跳
* 在非常靠近板子前沿的位置起跳，就像在蜂窝中做极限跳一样
* 熟悉空中扑的时机。你在扑之前并不会看到下一块板子从粘液里出来，但你要相信落地时会有板子出现。
